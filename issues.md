# Smart Clinic Management System - User Stories

## Admin Stories
- **Story**: As an **Admin**, I want to **add new doctors** to the system so that patients can book appointments with them.
- **Story**: As an **Admin**, I want to **view reports** on clinic performance so that I can make informed decisions.
- **Story**: As an **Admin**, I want to **manage user accounts** so that I can ensure system security.

## Doctor Stories
- **Story**: As a **Doctor**, I want to **view my daily appointments** so that I can manage my schedule efficiently.
- **Story**: As a **Doctor**, I want to **record patient prescriptions** so that the pharmacy can dispense medication.
- **Story**: As a **Doctor**, I want to **set my availability** so that patients know when I am free.

## Patient Stories
- **Story**: As a **Patient**, I want to **search for doctors by specialty** so that I can find the right care.
- **Story**: As a **Patient**, I want to **book an appointment** so that I can visit the clinic at a convenient time.
- **Story**: As a **Patient**, I want to **view my consultation history** so that I can track my health records.
